# External Sources

The **External Sources** tab lets you expand your chatbot's knowledge base beyond your own WordPress site by adding URLs from anywhere on the web — for example, a public documentation site, a help center, or a trusted reference. This is useful when the best answer to a question lives somewhere outside your own content.

## Accessing External Sources

Go to **Settings > Data Source** and click the **External Sources** tab.

## Adding an External Source

![Add External Source form with title, URL, and description fields](/images/page69_1.png)

The **Add External Source** section is where you register a new external URL as part of your chatbot's knowledge base.

Fill in the following fields:

### Source Title *(Required)*

A short, recognizable title for the source. For example, *"Documentation"* or *"Help Center"*. This helps you identify the source later in your list.

### Source URL *(Required)*

The full URL of the external source (for example, `https://example.com/knowledge-base`).

::: tip
Make sure the URL is publicly accessible so Zeno can read its content.
:::

### Context or Description *(Optional)*

A short description explaining what this source contains or when the chatbot should use it. Adding this helps the AI choose the right source when answering.

## Adding the Source

Once you've filled in the required fields, click the **Add Source** button to add this URL to your chatbot's knowledge base.

::: warning
The **Add Source** button will remain grayed out until you have filled in the required fields. It becomes active only when a valid title and URL are entered.
:::

Your chatbot can now draw from the external URLs you've added alongside your internal content.
