# Chat Settings

The **Chat Settings** section is where you customize how the floating chat widget looks and behaves on your website. From the agent's name and avatar to the colors of every message bubble, this is where you shape the visual identity of your chatbot and control how it greets, engages, and interacts with your visitors.

## Accessing Chat Settings

Click the **Settings** tab in the top-right corner of your Zeno Chatbot page. From the left-hand settings menu, click **Chat Settings**. This is the third tab in the list.

## Widget Visibility & Agent Identity

### Enable Floating Chat Widget

![Enable Floating Chat Widget toggle and Agent Name field](/images/page36_1.png)

Use this toggle to turn the floating chat widget on or off.

- When **enabled**, the widget appears on every page of your website.
- Turn it **off** to hide the widget entirely without removing any of your configuration.

The current status is shown on the right (*Enabled* or *Disabled*).

### Agent Name

Enter the display name that will appear as your chat agent. This is shown in the chat header and next to each message the chatbot sends. The default is **Support Agent**, but you can change it to anything that fits your brand — for example, *"Ask Zoe"* or *"Help Desk"*.

### Agent Avatar URL

![Agent Avatar URL field and Header Color picker](/images/page37_1.png)

Provide an image to use as the agent's avatar. Click the **Add Media** button to open your WordPress media library, where you can either select an existing image or upload a new one. A clear, square image works best for avatars.

## Color Customization

![Header Color and Send Button Color settings](/images/page38_1.png)

| Setting | Description | Default |
|---------|-------------|---------|
| **Header Color** | Background color of the chat header bar (where the agent name appears). | `#2563eb` |
| **Send Button Color** | Color of the send button in the chat input area. | `#2563eb` |
| **Bubble Background Color** | Background color for neutral system bubbles (system notifications or informational messages). | `#2563eb` |
| **Agent Chat Background** | Background color of the agent's message bubbles. | `#f3f4f6` |
| **Agent Chat Message Color** | Text color inside the agent's message bubbles. | `#1f2937` |

![Agent Chat Background and Agent Chat Message Color](/images/page39_1.png)
| **Customer Chat Background** | Background color of the customer's message bubbles. | `#2563eb` |
| **Customer Chat Message Color** | Text color inside the customer's message bubbles. | `#ffffff` |

![Customer Chat Background and Customer Chat Message Color](/images/page40_1.png)

Click the color swatch next to any setting to open the color picker, or enter a hex code directly.

## Chat Behavior

### Show Online Indicator

![Show Online Indicator toggle and Online Indicator Text field](/images/page41_1.png)

Use this toggle to show or hide an availability badge on the chat widget. When **enabled**, visitors can see at a glance whether you are online. The current status is shown on the right (*Enabled* or *Disabled*).

### Online Indicator Text

The message displayed next to the online badge when the indicator is enabled. The default is **"We're online"**, but you can customize it to anything like *"Usually replies in minutes"* or *"Live support available"*.

### Greeting Message

![Greeting Message and Support Ticket Link fields](/images/page42_1.png)

The first message visitors see when the chat opens. This welcomes users and sets the tone for the conversation. The default is **"Hi there! How can we help?"** — you can customize it to match your brand voice.

### Support Ticket Link

A link to your support ticket submission page. If a visitor needs help that the chatbot can't provide, they can be directed here. Enter a full URL such as `https://example.com/support`.

### Contact Us Page Link

![Contact Us Page Link and Auto Open Chat toggle](/images/page43_1.png)

A link to your Contact Us page. This gives visitors an alternative way to reach you outside of the chat. Enter a full URL such as `https://example.com/contact`.

### Auto Open Chat

Use this toggle to control whether the chat widget opens automatically after a visitor lands on your site. When **enabled**, the chat will pop open on its own after a configured delay. The current status is shown on the right (*Enabled* or *Disabled*).

### Auto Open Delay (ms)

![Auto Open Delay field and Save Settings button](/images/page44_1.png)

The delay before the chat opens automatically, in milliseconds. The default is **3000** (3 seconds). Raise the value for a gentler experience, or lower it to engage visitors sooner.

---

## Saving Your Settings

Once you have customized the widget's appearance and behavior to your liking, click the **Save Settings** button at the bottom-right of the page to apply your changes.

Your chat widget will now reflect the new name, colors, greeting, and behavior across your entire site.
