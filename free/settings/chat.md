# Chat Settings

::: info Free Version
This page covers the Free Version. [View Pro docs →](/settings/chat)
:::

The **Chat Settings** section is where you customize how the floating chat widget looks and behaves on your website. From the agent's name and avatar to the colors of every message bubble, this is where you shape the visual identity of your chatbot.

## Accessing Chat Settings

![Chat settings page](/images/free/page35_1.png)

Click the **Settings** tab in the top-right corner of your Zeno Chatbot page. From the left-hand settings menu, click **Chat Settings**. This is the third tab in the list.

## Configuring Chat Settings

### Enable Floating Chat Widget

![Widget enable toggle](/images/free/page36_1.png)

Use this toggle to turn the floating chat widget on or off. When **enabled**, the widget appears on every page of your website. Turn it off to hide the widget entirely without removing any of your configuration. The current status is shown on the right (*Enabled* or *Disabled*).

### Agent Name

![Agent name field](/images/free/page36_2.png)

Enter the display name that will appear as your chat agent. This is shown in the chat header and next to each message the chatbot sends. The default is **Support Agent**, but you can change it to anything that fits your brand — for example, *"Ask Zoe"* or *"Help Desk"*.

### Agent Avatar URL

![Agent avatar URL field](/images/free/page37_1.png)

Provide an image to use as the agent's avatar. Click the **Add Media** button to open your WordPress media library, where you can either select an existing image or upload a new one. A clear, square image works best for avatars.

### Header Color

![Header color picker](/images/free/page37_2.png)

Sets the background color of the chat header bar (the top portion of the chat widget where the agent name appears). Click the color swatch to open the color picker, or enter a hex code directly. The default is `#2563eb`.

### Send Button Color

![Send button color](/images/free/page38_1.png)

Sets the color of the send button in the chat input area. The default is `#2563eb` to match the header.

### Bubble Background Color

![Bubble background color](/images/free/page38_2.png)

Sets the background color used for neutral system bubbles, such as system notifications or informational messages. The default is `#2563eb`.

### Agent Chat Background

![Agent chat background](/images/free/page39_1.png)

Sets the background color of the agent's message bubbles (the messages your chatbot sends). The default is `#f3f4f6`, a soft neutral gray.

### Agent Chat Message Color

![Agent message color](/images/free/page39_2.png)

Sets the text color used inside the agent's message bubbles. The default is `#1f2937`, a dark near-black for strong readability.

### Customer Chat Background

![Customer chat background](/images/free/page40_1.png)

Sets the background color of the customer's message bubbles (the messages your visitor sends). The default is `#2563eb`.

### Customer Chat Message Color

![Customer message color](/images/free/page40_2.png)

Sets the text color used inside the customer's message bubbles. The default is `#ffffff` (white) for contrast against the blue bubble.

### Show Online Indicator

![Online indicator toggle](/images/free/page41_1.png)

Use this toggle to show or hide an availability badge on the chat widget. When **enabled**, visitors can see at a glance whether you are online. The current status is shown on the right (*Enabled* or *Disabled*).

### Online Indicator Text

![Online indicator text](/images/free/page41_2.png)

The message displayed next to the online badge when the indicator is enabled. The default is *"We're online"*, but you can customize it — e.g., *"Usually replies in minutes"* or *"Live support available"*.

### Greeting Message

![Greeting message field](/images/free/page42_1.png)

The first message visitors see when the chat opens. This welcomes users and sets the tone for the conversation. The default is *"Hi there! How can we help?"* — customize it to match your brand voice.

### Support Ticket Link

![Support ticket link](/images/free/page42_2.png)

A link to your support ticket submission page. If a visitor needs help that the chatbot can't provide, they can be directed here. Enter a full URL such as `https://example.com/support`.

### Contact Us Page Link

![Contact us link](/images/free/page43_1.png)

A link to your Contact Us page. This gives visitors an alternative way to reach you outside of the chat. Enter a full URL such as `https://example.com/contact`.

### Auto Open Chat

![Auto open toggle](/images/free/page43_2.png)

Use this toggle to control whether the chat widget opens automatically after a visitor lands on your site. When **enabled**, the chat will pop open on its own after a configured delay. The current status is shown on the right (*Enabled* or *Disabled*).

### Auto Open Delay (ms)

![Auto open delay field](/images/free/page44_1.png)

The delay before the chat opens automatically, in milliseconds. The default is **3000** (3 seconds). Raise the value for a gentler experience, or lower it to engage visitors sooner.

## Saving Your Settings

![Save settings button](/images/free/page44_2.png)

Once you have customized the widget's appearance and behavior to your liking, click the **Save Settings** button at the bottom-right of the page to apply your changes.

Your chat widget will now reflect the new name, colors, greeting, and behavior across your entire site.
