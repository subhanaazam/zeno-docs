# Resources

::: info Free Version
This page covers the Free Version. [View Pro docs →](/settings/data-source/resources)
:::

The **Resources** tab lets you upload images or PDF files as additional context for your AI chatbot. This is useful when your knowledge lives in documents that aren't published as web pages — product manuals, policy PDFs, diagrams, or any file you want your chatbot to be able to reference.

## Uploading Resources

![Resources tab](/images/free/page70_1.png)

The **Resources** section is where you upload files to add to your chatbot's knowledge base. The description reads *"Upload images or PDFs to provide additional context to your AI chatbot."*

![Upload section](/images/free/page70_2.png)

- **Select Files** — Click this button to open a file picker and choose one or more files to upload. When no resources have been uploaded yet, you'll see a placeholder message: *"No resources uploaded yet."*

## Allowed File Types

![File types list](/images/free/page71_1.png)

Zeno Chatbot accepts the following file formats as resources:

- **JPG**
- **PNG**
- **GIF**
- **WebP**
- **PDF**

::: tip
Choose files that contain information relevant to the questions your visitors are likely to ask. For example, a PDF of your shipping policy can help the chatbot answer questions about delivery times even if that info isn't on your website.
:::

Once your files are uploaded, your chatbot can reference them when answering visitor questions.
