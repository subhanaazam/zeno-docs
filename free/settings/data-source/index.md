# Data Source

::: info Free Version
This page covers the Free Version. [View Pro docs →](/settings/data-source/)
:::

The **Data Source** section is where you build the knowledge base your chatbot draws from when answering questions. Instead of relying only on the AI model's general knowledge, Zeno Chatbot can pull from your own website content, external URLs, uploaded files, and custom notes — so answers stay accurate, on-brand, and specific to your business.

## Accessing Data Source

![Data source page](/images/free/page60_1.png)

Click the **Settings** tab in the top-right corner of your Zeno Chatbot page. From the left-hand settings menu, click **Data Source**. This is the seventh and final tab in the list.

## Data Source Tabs

![Data source tabs](/images/free/page61_1.png)

At the top of the Data Source page, you'll find **five tabs** that represent different types of knowledge sources you can feed to your chatbot:

| Tab | Description |
|-----|-------------|
| [Internal Content](./internal-content) | Index your own WordPress posts, pages, sitemap, and WooCommerce products |
| [External Sources](./external-sources) | Add external URLs to expand your chatbot's knowledge beyond your own site |
| [Resources](./resources) | Upload images or PDFs to provide additional context |
| [Knowledge Notes](./knowledge-notes) | Add custom markdown or plain text notes directly as AI context |
| **Automatic Builder** | *Upcoming — an automated knowledge base builder coming in a future release* |

Click any of the links above to learn more about that specific knowledge source.
