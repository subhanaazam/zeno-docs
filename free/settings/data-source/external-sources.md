# External Sources

::: info Free Version
This page covers the Free Version. [View Pro docs →](/settings/data-source/external-sources)
:::

The **External Sources** tab lets you expand your chatbot's knowledge base beyond your own WordPress site by adding URLs from anywhere on the web — for example, a public documentation site, a help center, or a trusted reference. This is useful when the best answer to a question lives somewhere outside your own content.

## Add External Source

![External sources tab](/images/free/page69_1.png)

The **Add External Source** section is where you register a new external URL as part of your chatbot's knowledge base. The description reads *"Add external URLs with title and context to expand your knowledge base."*

To add a new external source, fill in the following fields:

- **Source Title** — A short, recognizable title for the source. For example, *"Documentation"* or *"Help Center"*. This helps you identify the source later in your list.
- **Source URL** — The full URL of the external source (e.g., `https://example.com/knowledge-base`). Make sure the URL is publicly accessible so Zeno can read its content.
- **Context or description of this source (optional)** — A short description explaining what this source contains or when the chatbot should use it. Adding it helps the AI choose the right source when answering.

Once you've filled in the fields, click the **Add Source** button to add this URL to your chatbot's knowledge base.

::: warning
The **Add Source** button will remain grayed out until you have filled in the required fields. It becomes active only when a valid title and URL are entered.
:::

That's it! Your chatbot can now draw from the external URLs you've added alongside your internal content.
